# Prachi Kumari - Portfolio

## Overview

This is a personal portfolio website for Prachi Kumari, a student web developer. The portfolio showcases a minimalist design philosophy with clean interfaces and user-friendly navigation. The project is a static website built with HTML and CSS, focusing on simplicity and responsive design principles.


**November 16, 2025**: Initial portfolio build completed
- Created responsive HTML structure with Header, Hero, About, Skills, and Footer sections
- Implemented CSS styling with mobile-first responsive design
- Added hamburger menu for mobile navigation (checkbox-based toggle)
- Created skills grid with 6 skill cards (HTML, CSS, JavaScript, Java, Git/GitHub, Responsive Design)
- Configured Python HTTP server workflow to serve static files on port 5000
- All sections fully responsive: mobile (1 column), tablet (2 columns), desktop (3 columns)

**Technology Stack**: Pure HTML5 and CSS3
- Static website with no framework dependencies
- Client-side only implementation with no backend requirements
- Responsive design using CSS media queries and flexbox layout


**Layout Structure**:
- Single-page application (SPA) style with anchor-based navigation
- Four main sections: Home (Hero), About, Skills, Contact
- Container-based layout system with max-width constraint (1200px) for content centering
- Flexbox-based header with logo and navigation components

**Navigation System**:
- Sticky header that remains visible during scroll
- Hamburger menu toggle for mobile responsiveness (checkbox-based implementation)
- Smooth scrolling between page sections using anchor links

**Styling Approach**:
- CSS reset using universal selector for consistent cross-browser rendering
- System font stack for native look and feel across different platforms
- Subtle shadows and minimal color palette for clean aesthetic
- Z-index layering for sticky header positioning
